# My Portfolio

A modern, responsive personal portfolio website built with Next.js 16, featuring smooth animations and a sleek dark theme.

## Features

- **Modern Design** - Clean, dark-themed UI with glassmorphism effects
- **Smooth Animations** - Powered by Framer Motion with parallax mouse tracking
- **Fully Responsive** - Optimized for all screen sizes
- **Interactive Sections** - Hero, Works, Skills, Contact, and Footer components
- **Resume Viewer** - Built-in PDF resume viewer with download option

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) with App Router
- **Language:** TypeScript
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jittub45/My_Portfolio-.git
   cd My_Portfolio-/portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
